  • 3th Semester Mainproject: LarsQuizmas

    Lars Quizmas is a Web based Quiz Application by Christoph Stiller, Lars Assmus, Kevin Lee, Holger Schad and Marco Berg

    It was Build as 1 week project to master Scrum and to Develop a Quiz app.
    Lars Quizmas is Build on top of Lamest Webserver*. Because of this we were able to Code fast and Clean in C# and our project was by far the best of all projects in this Course.

    Quizzard is the successor of LarsQuizmas Created by Christoph Stiller, Lars Assmus, Paul Schaefer and Sinan Ayhan

Quizzard is more dynamic has more Fetures like a Course system, LiveQuiz, IntelliQuiz and more. Quizzard is still in Development.
